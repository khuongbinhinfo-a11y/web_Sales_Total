🔐 Shared Secret cho AI-app

Dùng mục này để lấy đúng shared secret đang chạy trên Web tổng production rồi chuyển cho AI-app qua kênh secure.

Hướng dẫn tạo/đổi secret key (không cần thao tác Vercel)

  1. Chọn profile key: shared, web, hoặc desktop.
  2. Dùng nút Tạo và lưu ngay để hoàn tất trong 1 click.
  3. Khuyến nghị: app có bản web + desktop nên tách 2 key riêng để dễ thu hồi/rà soát.
  4. Header chuẩn: x-ai-app-key; profile tùy chọn: x-ai-app-profile.
