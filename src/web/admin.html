🔐 Shared Secret cho AI-app

Dùng mục này để lấy đúng shared secret đang chạy trên Web tổng production rồi chuyển cho AI-app qua kênh secure.

Hướng dẫn tạo/đổi secret key (rõ ràng từng bước)

  1. Bấm "Tạo key mẫu 32 ký tự" để sinh key mới (local, không tự lưu).
  2. Sao chép lệnh Vercel được tạo sẵn và chạy trên máy deploy.
  3. Deploy lại production.
  4. Bấm "Hiện shared secret" để đối chiếu key đang chạy thực tế.
