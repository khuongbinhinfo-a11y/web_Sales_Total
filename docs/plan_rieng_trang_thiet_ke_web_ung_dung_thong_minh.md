# PLAN UX/UI - NHANH THIET KE WEB (THUC THI NGAY)

Muc tieu: Dev doc la lam ngay theo dung y da chot.
Pham vi: Chi uu tien nhanh Thiet ke Web, khong thay doi cau truc web hien co.

## 1) Scope cung

1. Khong thay doi cau truc web hien co.
2. Khong doi logic he thong dang chay.
3. Khong quay lai huong 3-4 card lon phan loai o dau trang.
4. Trang nay co 2 nhanh con:
   - Nhanh 1: Thiet ke Web (lam chi tiet ngay)
   - Nhanh 2: Web mau trien khai nhanh (tam thoi chi tom tat + dieu huong)

## 2) Nhanh 1 - Thiet ke Web (lam truoc)

### 2.1 Hero

1. Video hero bat buoc: hero-web-brand.mp4.
2. Text hero chi giu 1 tieu de chinh.
3. Tieu de dat o chan video, van nam trong vung video.
4. Khong dat mo ta dai, khong nhieu text.

### 2.2 Khoi chinh demo theo nganh

1. Bo hoan toan khoi 4 card lon phan loai nhu cau.
2. Dung mo hinh card nho dieu huong + 1 card lon preview.
3. Danh sach card nho mac dinh:
   - Web Gioi Thieu
   - Dich Vu
   - Web Ban Hang
   - Shop
   - Web Catalogue
   - B2B
   - Landing Page
   - Dat Lich
4. Neu thieu nganh thi bo sung theo mo hinh cu.
5. Giai doan nay chua can viet chi tiet noi dung tung nganh.

## 3) Interaction bat buoc

1. Desktop: hover card nho -> card lon doi preview.
2. Mobile: tap card nho -> card lon doi preview.
3. Card dang chon phai co active state ro rang.
4. CTA toi gian trong nhanh nay: Xem mau website.

## 4) Nhanh 2 - Web mau trien khai nhanh (tam thoi)

1. Chi de 1 section tom tat ngan.
2. Muc tieu: dieu huong sang nhanh rieng Web mau.
3. Chua viet flow chi tiet trong tai lieu nay.
4. Chi gom cac noi dung hien co, khong mo rong feature.

## 5) Nghiem thu 1 vong (done criteria)

1. Tai lieu the hien ro 2 nhanh con.
2. Hero dung dung video hero-web-brand.mp4.
3. Hero chi con 1 tieu de, dat o chan video.
4. Khong con khoi 4 card lon o dau trang.
5. Mo hinh card nho -> preview card lon hoat dong ro cho desktop/mobile.
6. Nhanh Web mau chi o muc tom tat + dieu huong, khong di chi tiet.
7. Khong co yeu cau nao lam thay doi cau truc web hien co.

---

## Ban giao cho dev

Dev chi can bam dung 5 muc tren de thuc hien UI nhanh Thiet ke Web.
Neu can mo rong chi tiet tung nganh hoac flow Web mau, se tach tai lieu phase tiep theo.
